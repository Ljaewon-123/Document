## node 접근방법

DOM node에 접근하는 방법은 여러 가지가 있지만, 가장 일반적인 방법은 JavaScript를 이용하여 document 객체를 통해 접근하는 것 document 객체는 현재 문서를 나타내며, 문서 내의 모든 DOM node에 접근할 수 있는 메서드와 프로퍼티를 제공함.
